# Image-to-PDF

A python script that converts a list of images to a single PDF file.

<br />

## How to run :

1) Clone this repository :

```
git clone https://github.com/kevinadhiguna/img-to-pdf.git
```

2) Install dependencies :

```
pip install -r requirements.txt
```

3) Run program :

```
python pdf-gen.py
```
